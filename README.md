# whatsapp-bot
Automação Java com Selenium WebDriver para envio de mensagens no WhatsApp Web. O projeto utiliza lógica de RPA para interagir com o DOM, incluindo busca de contactos específicos e envio em massa para conversas ativas. Demonstra o uso de esperas dinâmicas (WebDriverWait) e seletores avançados para manipulação de elementos web complexos.
